# Flutter Hello World App

## Overview

This project is a basic Flutter application that displays a simple "Hello Flutter!" message on the screen. It serves as an introduction to Flutter development and demonstrates the structure of a minimal Flutter app.

---

## Features

* Displays a centered text message
* Includes an AppBar with a title
* Simple and clean user interface
* Beginner-friendly example

---

## Technologies Used

* Flutter
* Dart
* Material UI components

---

## Project Structure

* `main.dart`

  * Contains the entire application logic
  * Includes:

    * `MyApp`: Root widget of the application

---

## How It Works

1. The app starts by running the `MyApp` widget.
2. A `MaterialApp` widget is used as the root.
3. The home screen contains:

   * An AppBar with the title "Hello World"
   * A centered text widget displaying "Hello Flutter!"
4. The text is styled with a font size of 24.

---

## How to Run

1. Install Flutter on your system.
2. Create a new Flutter project or replace the content of `main.dart` with this code.
3. Run the following command:

   ```
   flutter run
   ```
4. Launch the app on an emulator or physical device.
